# 📊 Data Analysis Dashboard

A simple data visualization web app built using Flask, Pandas, and Matplotlib.  
It analyzes a dataset and displays charts like salary distribution, job roles, and experience levels.

---

## 🚀 Features
- 📈 Salary distribution histogram
- 🌍 Top company locations
- 💼 Top job roles
- 🧑‍💻 Experience level pie chart
- 📋 Raw and cleaned dataset preview

---

## 🛠️ Technologies Used
- Python (Flask)
- Pandas
- Matplotlib
- HTML, Bootstrap
